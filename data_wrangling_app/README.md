# 📊 Data Wrangling & Analytics Platform (DWAP)

**Profit Projects Online Virtual Assistance**

DWAP is a high-performance, enterprise-grade data processing and machine learning application. It provides an intuitive interface for data cleaning, visualization, and advanced predictive analytics using state-of-the-art algorithms.

## 🌟 Premium Features

### 🧪 Advanced Machine Learning & AI
- **XGBoost Model Lake**: Save and manage countless model versions. Browse your model history and load specific versions for targeted predictions.
- **Collective Intelligence**: Run asynchronous predictions across multiple models in the "Lake" simultaneously to gain consensus insights.
- **Ensemble Workflows**: Integrated pipeline for **Voting** and **Stacking** strategies using an extensive library of custom-coded machine learning and neural network models.
- **Auto-ML Evaluation**: Automatic model selection and performance benchmarking on processed datasets.

### ⚡ Extreme Performance
- **FasterPython Integration**: Core ML components (Neural Networks, Data Cleaning) are optimized with high-performance Python patterns, including local variable binding and reduced attribute lookups.
- **Asynchronous Processing**: Leverages `asyncio` for non-blocking model training, I/O operations, and parallel inference, ensuring a smooth and responsive UI.

### 📤 Multi-Format Engineering
- **Intelligent Loading**: Support for CSV, Excel (Multi-sheet), JSON (Normalized), PDF (Extract), DOCX (Paragraph analysis), and adaptive Text parsing.
- **Advanced Wrangling**: Automated duplicate removal, strategic missing value handling (Mean/Mode/Drop), text standardization, and IQR-based outlier detection.

### 📊 Professional Visualizations
- **Interactive Dashboards**: Real-time correlation heatmaps, distribution analysis, and time-series forecasting.
- **Custom Plot Engine**: Plotly-based dynamic charts with professional styling.

### 💾 Enterprise Data Management
- **Searchable Database**: SQLite-backed storage for datasets, metadata, and processing history.
- **Sharing & Reporting**: Professional HTML email reports and multi-format exports (CSV, XLSX, JSON).

## 🚀 Installation & Setup

### Prerequisites
- Python 3.9+
- pip

### Quick Start
1. **Clone & Enter**
   ```bash
   git clone <repository-url>
   cd data_wrangling_app
   ```
2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
3. **Launch**
   ```bash
   streamlit run app.py
   ```

## 🛠️ Architecture Overview

- **`app.py`**: Main application logic and UI orchestration.
- **`utils/ml_integration.py`**: XGBoost Model Lake and core Neural Network implementations.
- **`utils/ensemble_engine.py`**: Logic for combining multiple custom models.
- **`utils/custom_ml_wrapper.py`**: Standardization bridge for disparate custom ML scripts.
- **`utils/data_processor.py`**: Optimized data cleaning and transformation engine.
- **`utils/visualizations.py`**: Plotly-based visualization suite.
